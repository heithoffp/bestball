Privacy Policy

Last updated: April 7, 2026

Best Ball Exposures ("we", "our", "us") operates the Best Ball Exposures web application at bestballexposures.com and the Best Ball Exposures Chrome extension. This policy explains what data we access and how it is handled.

What the Chrome Extension Does

The Best Ball Exposures Chrome extension enhances your drafting experience on supported platforms (Underdog Fantasy, DraftKings) by displaying your portfolio exposure data alongside the draft board. It reads publicly visible draft-room information from these sites to provide real-time context.

Data Collection

We do not sell your personal data to third parties.

Chrome Extension

The Chrome extension does not collect, transmit, or store any data externally. Specifically:

  • No personal information is collected — no names, emails, or account credentials.
  • No browsing history is tracked or recorded.
  • No data is sent to external servers. All processing happens locally in your browser.

Web Application

If you create an account on the web app at bestballexposures.com, we collect:

  • Email address — for authentication purposes.
  • Portfolio data — rosters and settings you choose to sync to the cloud via Supabase.

This data is stored securely in Supabase and is used solely to provide the service. Without an account, all data remains local to your browser via IndexedDB.

Local Storage

The extension uses Chrome's local storage API (chrome.storage.local) to save your preferences and cached portfolio data on your device. This data never leaves your browser.

Host Permissions

The extension requests access to the following domains solely to read draft-room data and provide portfolio context:

  • app.underdogfantasy.com
  • api.underdogfantasy.com
  • stats.underdogfantasy.com
  • www.draftkings.com
  • api.draftkings.com

No data from these sites is collected, stored remotely, or shared with third parties.

Third-Party Services

  • Supabase — authentication and cloud storage (optional)
  • Vercel Analytics — anonymous, cookie-free page-view metrics (no personal data)

Changes to This Policy

We may update this policy as the product evolves. Changes will be posted on this page with an updated date.
