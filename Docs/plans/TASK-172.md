# TASK-172: Feature gating review — evaluate guest "taste" experience for conversion

**Status:** Draft
**Priority:** P2

---

## Objective
Evaluate whether the current guest tier (Dashboard + Exposures only) provides enough value to convert users when competing against free tools that show everything. Analyze what free competitors (from TASK-163) offer for free vs. what our guest tier shows. Consider alternatives: (a) expand guest tier to include more tabs with limited functionality, (b) allow sample-data preview of Pro tabs, (c) time-limited full-access trial, (d) keep current gating but improve the upgrade prompts. Produce a recommendation with rationale. Addresses findings F-017 and F-020 from the systems model.

## Dependencies
TASK-163 (need competitive landscape to assess relative value of free tier)

## Open Questions
- Is the beta access period (TASK-161) sufficient to address this for launch?
- If we expand guest tier, which tabs add the most conversion value?
- How do we prevent "good enough for free" if we show too much?
