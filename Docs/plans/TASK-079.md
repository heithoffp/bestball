# TASK-079: Ensure color independence for trend indicators

**Status:** Draft
**Priority:** P4

---

## Objective
ADP trend indicators throughout the app (ExposureTable sparklines, ADP Tracker trend values, Dashboard) use color alone (green for rising, red for falling) to convey directional information. UI/UX Guide section 11 requires: "Never rely on color alone to convey information. Pair trend colors with directional icons (arrow up/down)." Ensure all trend indicators pair color with an icon or text indicator for accessibility compliance.

## Dependencies
None
