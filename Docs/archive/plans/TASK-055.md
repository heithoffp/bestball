# TASK-055: Roster-level strategy scorecard — sortable table across all rosters

**Status:** Draft
**Priority:** P2

---

## Objective

Create a sortable, filterable table that shows every roster as a single row with key summary columns: archetype path (RB/QB/TE), CLV, uniqueness score, and strategy viability score. Currently these metrics exist in Roster Viewer but require clicking into each roster individually. A single-view scorecard lets the user spot outliers — uniquely bad or uniquely good rosters — across their whole book at a glance and then drill into the ones that warrant attention.

## Dependencies

None

## Open Questions

- Should this be a new tab, or an enhanced view within Roster Viewer (e.g., a "Summary" mode toggle)?
- Which columns are most valuable: archetype, CLV average, uniqueness rank, strategy score, or others?
- Should clicking a row navigate into the full Roster Viewer for that entry?
